# Publish Guide

This is a static website. You can publish the whole `honor-heroes-site` folder on any static hosting platform.

Fastest option:

1. Open https://app.netlify.com/drop
2. Drag the `honor-heroes-site` folder into the page
3. Netlify will generate a public URL after upload

Other options:

- Vercel: create a new project and upload/import this folder.
- Cloudflare Pages: create a Pages project and upload this folder.
- GitHub Pages: push this folder to a repository and set Pages source to the repository root.

Entry file:

```text
index.html
```
